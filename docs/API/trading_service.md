# 交易服务 API 文档

## 外部自选股 API (192.168.66.143:8000)

项目使用外部 API 服务管理自选股票。

**Base URL**: `http://192.168.66.143:8000`

### 接口列表

#### 1. 新增自选股票

| 项目 | 说明 |
|------|------|
| **Endpoint** | `POST /ts_code` |
| **请求体** | `{"ts_code": "600519"}` |
| **响应 200** | `{"id": 1, "ts_code": "600519", "created_at": "2024-01-15T10:30:00"}` |

#### 2. 删除自选股票

| 项目 | 说明 |
|------|------|
| **Endpoint** | `DELETE /ts_code/{ts_code}` |
| **响应 200** | `{}` |

#### 3. 获取所有自选股票实时行情

| 项目 | 说明 |
|------|------|
| **Endpoint** | `GET /stock/realtime` |
| **响应 200** | `{"data": [...], "update_time": "2024-01-15T10:30:00"}` |

#### 4. 启动 WebSocket 推送

| 项目 | 说明 |
|------|------|
| **Endpoint** | `POST /ws/start` |

#### 5. 停止 WebSocket 推送

| 项目 | 说明 |
|------|------|
| **Endpoint** | `POST /ws/stop` |

#### 6. 查看 WebSocket 状态

| 项目 | 说明 |
|------|------|
| **Endpoint** | `GET /ws/status` |
